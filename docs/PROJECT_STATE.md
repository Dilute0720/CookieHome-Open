## 已知问题

- 当前登录已支持成员账号，但仍保留无密码哈希用户可用家庭密码登录的兼容逻辑。
- 权限已覆盖主要管理操作，但后续仍可继续细化到更丰富的角色，例如采购员、做饭人等。
- HEIC/HEIF 手机照片会尝试由服务器图片库解码；如果服务器 sharp/libvips 不支持该格式，表单会提示改用 JPG/PNG 或将 iPhone 相机格式设为“最兼容”。
- 食材名称已支持历史建议和大小写/空白归一，但“土豆/马铃薯”这类同义词仍需要后续别名系统处理。
- 菜品标签已支持历史建议和空白归一，但暂未提供标签批量管理/重命名页面。
- 智能推荐目前是规则引擎首版，季节、天气和更长期的重复控制仍待增强。
- 文件存储当前为本地开发模式，已预留 S3 兼容对象存储接口但尚未接入真实凭据和 SDK。
- 文件存储当前保持本地 `public/uploads`，OSS driver 只预留接口，尚未接入真实 SDK。
- 家庭时间轴图片已接入当前本地存储和压缩链路，并支持编辑时调整顺序；后续可继续增强为拖拽排序和封面图选择。
- 博客图文小记首版支持发布和展示图片/标签，暂未支持编辑已发布小记的图片和标签。
- 小动静的菜单撤回目前只恢复待办状态并删除自动记录；库存扣减没有独立流水账，暂不自动回补原批次库存。
